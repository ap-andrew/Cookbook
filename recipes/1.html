Topic name:
Environment Configuration
Level:
Intermediate
Description:

Map an Apigee environment to a custom domain

Doc:

curl -u user:pass https://api.enterprise.apigee.com/v1/o/ORG/e/ENV/virtualhosts/default > vh.json

Now look at "vh.json":

{
"hostAliases" : [ "ORG-ENV.apigee.net" ],
"interfaces" : [ ],
"name" : "default",
"port" : "80"
}

Now edit the file to add another host alias:

{
"hostAliases" : [ "ORG-ENV.apigee.net", "api.your.name.here.com" ],
"interfaces" : [ ],
"name" : "default",
"port" : "80"
}

And now replace it:

curl -u user:pass https://api.enterprise.apigee.com/v1/o/ORG/e/ENV/virtualhosts/default -H "Content-Type: application/json" -X PUT -T vh.json
